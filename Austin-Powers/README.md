# Austin Powers

Given an integer n greater than or equal to 0, return whether it is a power of two.

## Example 1

### Input

```typescript
n = 0
```

### Output

```typescript
false
```

## Example 2

### Input

```typescript
n = 1
```

### Output

```typescript
true
```

### Explanation

2^0 = 1

## Example 3

### Input

```typescript
n = 2
```

### Output

```typescript
true
```

### Explanation

2^1 = 2
