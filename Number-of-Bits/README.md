# Number of bits

Given an integer `n` greater than or equal to `0`, return the number of 1 bits in `n`.

## Example 1

### Input

```javascript
n = 0
```

### Output

```javascript
0
```

## Example 2

### Input

```javascript
n = 1
```

### Output

```javascript
1
```

## Example 3

### Input

```javascript
n = 2
```

### Output

```javascript
1
```

### Explanation

2 is `10` in binary.

## Example 4

### Input

```javascript
n = 3
```

### Output

```javascript
2
```

### Explanation

3 is `11` in binary.

## Example 5

### Input

```javascript
n = 4
```

### Output

```javascript
1
```

### Explanation

4 is `100` in binary.
