# Longest consecutive duplicate string

Given a string `s`, return the length of the longest substring with same characters.

For example, given `abcccdda`, return 3 since it's the length of `ccc`.

## Example 1

### Input

```python
s = "abbbba"
```

### Output

```python
4
```

## Example 2

### Input

```python
s = "aaabbb"
```

### Output

```python
3
```
