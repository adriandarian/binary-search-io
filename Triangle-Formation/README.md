# Triangle formation

Given three integers `a`, `b`, and `c`, return whether the sum of the smallest two numbers is greater than the largest.

## Example 1

### Input

```javascript
a = 4
b = 5
c = 6
```

### Output

```javascript
true
```

## Example 2

### Input

```javascript
a = 1
b = 2
c = 3
```

### Output

```javascript
false
```
