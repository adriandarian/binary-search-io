# Hip to Be Square

Given a sorted list of integers, square the elements and give the output in sorted order.

> Note: The integers can be 0 or negative.

## Example 1

### Input

```javascript
nums = [-9, -2, 0, 2, 3]
```

### Output

```javascript
[0, 4, 4, 9, 81]
```

## Example 2

### Input

```javascript
nums = [1, 2, 3, 4, 5]
```

### Output

```javascript
[1, 4, 9, 16, 25]
```
