# Taco Cat

Given a string `s`, return whether it is a palindrome. A palindrome is when the word is the same forwards and backwards.

For example, `"tacocat"` is a palindrome.

## Example 1

### Input

```javascript
s = "racecar"
```

### Output

```javascript
True
```

## Example 2

### Input

```javascript
s = "evilolive"
```

### Output

```javascript
True
```

## Example 3

### Input

```javascript
s = "palindrome"
```

### Output

```javascript
False
```
