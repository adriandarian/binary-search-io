# Merging two sorted lists

Given two sorted lists of integers, merge them into one large sorted list.

For example, given these two lists:

```javascript
lst0 = [5, 10, 15]
lst1 = [3, 8, 9]
```

Return `[3, 5, 8, 9, 10, 15]`.

## Example 1

### Input

```javascript
lst0 = [5, 10, 15]
lst1 = [3, 8, 9]
```

### Output

```javascript
[3, 5, 8, 9, 10, 15]
```
