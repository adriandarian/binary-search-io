# 137 Yonge St

Given a list of integers of length 3, determine whether it contains the number 1, 3, or 7.

## Example 1

### Input

```javascript
nums = [5, 1, 9]
```

### Output

```javascript
true
```

### Explanation

This array contains `1`

## Example 2

### Input

```javascript
nums = [8, 5, 9]
```

### Output

```javascript
false
```

### Explanation

This array doesn't contain `1`, `3`, or `7`.

## Example 3

### Input

```javascript
nums = [1, 3, 7]
```

### Output

```javascript
true
```

### Explanation

This array contains `1`, `3`, and `7`.
